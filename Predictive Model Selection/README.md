# Predictive Model Selection

This project builds predictive models for estimating body fat percentage using multiple regression. Several model selection techniques are compared and evaluated using predictive performance.

## Methods
- Multiple regression
- Forward selection
- Backward elimination
- Best subset selection
- Model comparison using AIC, BIC, and Cp
- RMSE evaluation on a test dataset

## Files
- `bodyfat_model_selection.R` – R script containing the analysis
- `fat.csv` – dataset used for prediction

## Tools
- R
- ggplot2
- olsrr
- leaps

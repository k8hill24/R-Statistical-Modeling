# Multiple Regression Analysis – Galapagos Islands

This project models species richness across Galapagos islands using multiple linear regression. 
Geographic and environmental variables are used to explain variation in species counts.

## Methods
- Multiple linear regression
- Residual diagnostics
- Leverage analysis
- Cook's distance
- Identification of influential observations

## Files
- `galapagos_multiple_regression.R` – R script containing the analysis
- `galapagos.csv` – dataset used for modeling

## Tools
- R
- ggplot2

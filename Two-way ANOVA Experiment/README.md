# Two-Way ANOVA Experiment

This project analyzes how timing and light intensity influence flower production using a two-way ANOVA model. Interaction effects between the experimental factors are evaluated and compared with additive models.

## Methods
- Exploratory data visualization
- Two-way ANOVA
- Interaction effects
- Additive vs interaction model comparison
- Extra sum-of-squares F-test
- Parallel regression model

## Files
- `flower_anova_analysis.R` – R script containing the analysis
- `case0901.csv` – dataset used in the experiment

## Tools
- R
- ggplot2
